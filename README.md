## 文档

> [主题文档](https://butterfly.js.org/)

## Quickstart

安装依赖:

```bash
# Node.js >= 18.0.0 is required.
yarn install
```

运行项目

```bash
yarn run server
```

打包项目

```bash
yarn run build
```
